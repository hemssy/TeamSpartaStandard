## 4-2 실습 과제 

**첫번째 섹션 (배너 섹션)**
  - 그룹
    - 가로 80%, 세로 160%
  - 아이템
    - 가로 100%, 세로 100%
    - 가로 스크롤
    - BannerCell 사용

**BannerCell**
  - titleLabel (UILabel), bgView (UIView)
  - titleLabel
    - font: 22, bold
  - bgView
  - cornerRadius: 16
  - backgroundColor: Purple


**두번째 섹션 (그리드 섹션)**
  - 높이 100으로 3열로 만들기


**GridCell**
  - gridLabel (UILabel)
  - 폰트: semibold 크기 18
  - Text 중앙 정렬
    

두 섹션의 데이터 타입은 String 배열을 사용하고 내용은 자유

키워드 힌트는 UICollectionViewCompositionalLayout 입니다!
